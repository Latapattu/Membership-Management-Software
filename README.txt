Membership Management Software
for CPS 406, Vojislav Misic
by Lahiru Atapattu, Jessica Ip, Dahee Kim, Noah Lattari, Sindy Wang

Skeleton of the membership application. Displays Login frame, where user can login or create a new account.
Input for login is currently "member", "coach", or "treasurer". Inputs will lead to the Member frame, Coach frame, and Treasurer frame respectively. 
Member frame has panels to view messages and reminders. Coach frame has panels to send announcements and messages, as well as option to view member log. Treasurer frame has panels to view unpaid expenses, prepaid fees, coaches, and the schedule for the club.
